Mark Dir as source? Resource Root at File | Settings | Directories. ?
Project > Directories

To make file references to a non-root folder in your project, you'll need to mark that directory as a Resource.
To do this, go to Settings > Directories and select your asset root folder from the right-hand panel.
Then mark it as Resource Root by clicking the button above the file browser.

From the Jetbrains documentation, a Resource Root enables WebStorm to complete relative paths to resources under the selected folder.
This means that you should use Resource Root on a folder that holds all the resources you would like to refer to (e.g. styles, scripts, ...).
